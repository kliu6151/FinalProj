## Instructions to setup and run project
Install Node.js

In Server: INSTALL:
npm install express
npm install mongoose
npm install nodemon

In Client: INSTALL:
npm install axios

To start it, first run the command:
node .\populate_db.js mongodb://127.0.0.1:27017/fake_so
